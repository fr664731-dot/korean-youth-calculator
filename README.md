# 청년 맞춤형 정책 금융 계산기

청년도약계좌, 청년 월세 특별지원 등 청년 정책 금융 상품의 예상 수익을 쉽게 계산할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- **청년도약계좌 계산기**: 월 납입액과 소득 수준에 따른 5년 후 만기 금액 계산
- **청년 월세 특별지원 확인기**: 월세 지원 자격 확인
- **2025년 1월 개정안 반영**: 최신 정부 정책 기준 적용
- **반응형 디자인**: 모바일/데스크톱 최적화
- **토스 스타일 UI**: 깔끔하고 직관적인 사용자 경험

## 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Charts**: Recharts
- **Icons**: Lucide React
- **Font**: Pretendard

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 빌드

```bash
npm run build
```

### 프로덕션 실행

```bash
npm start
```
